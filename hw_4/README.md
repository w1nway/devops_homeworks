# Docker Compose Конфигурация для Nginx и MySQL

## Инструкции по настройке

1. Запуск с помощью Docker Compose:
    ```sh
    docker-compose up -d
    ```

2. Веб-сервер Nginx:
    - Перейти по адресу `http://localhost:8080`

3. Для остановки сервисов выполните команду:
    ```sh
    docker-compose down
    ```
